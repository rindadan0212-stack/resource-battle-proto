## 進行(実ゲームと同等の骨格)
- **拠点(1章6日)**: ダンジョン / モルガンで働く(即金・淫乱↑) / 宿 / ショップ(日を消費しない) / **スキルツリー**。
- **押し引きダンジョン**: 1階ずつ潜り、勝つたび稼ぎ**プール**が連勝倍率で膨張→「奥へ進む/引き上げる」。HP/EP/**MP**は階持ち越し。3・5階=ボス雑魚。負ければプール全損+淫乱↑。
- **ショップ**: Gで武具/防具/淫具/消耗品(ポーション/聖水/力の薬/**マナ水**)を買う。
- **返済日(7日目)**: 章ボス→ノルマ納付。不足=利子繰越(最終章=破産)。全4章クリア→エンド(淫乱度分岐)。

## 戦闘 v3(リソース制 + MP + スキルツリー)
- **1手番=エネルギー(行動力)の予算内で複数行動**。行動は**攻撃 / スキル / 防御(1ターン1回) / どうぐ**。
- **スキル=エネルギー + MP の二重コスト**。MPは**探索(run)持ち越しの枯渇資源**(戦闘開始時+3のみ回復)=深追いすると気力も尽きる。マナ水(金)で回復。
- スキル7種: 強撃・崩し・見切り + **集中(自バフ)/威圧(敵ATK↓)/弱点看破(敵DEF↓)/深呼吸(EP↓)**。**敵デバフはスキル専用**、どうぐ(金)は自己強化・回復に領分分離。
- **興奮(EP)が上がるほど最大エネルギーが減少**(−⌊EP/12⌋)→手数減→**絶頂敗北(EP100)**への死のスパイラル。
- **レベルアップ=自動成長を廃止 → SP獲得 → スキルツリー(3系統26ノード)** でステ/スキル/パッシブを解放。クリアで約半分しか取れない=ビルド個性(剣=火力/体=生存/心=MP・術)。キーストーン「高揚」=EP50+でATK+20%(感じるほど強い)。淫気解放は無し。

## 「殴り合い脱却」の数値証明
下部の**オート対戦**で 攻撃連打AI / スキル連打AI / 賢いAI(EPレース管理・大技を見切り・デバフ)の勝率を比較。**賢い＞連打** ならリソース管理に意味がある。検証例: ボス雑魚(EPレース)で**賢い−連打 +55pt**、最終ボス魔人で**+33pt**、ビルド3種で勝率が有意に割れる(物理ボスは体の道が強い等)。
