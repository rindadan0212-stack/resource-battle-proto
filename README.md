# 借金返済ダンジョンRPG — 進行＆リソース制戦闘 検証プロトタイプ

ターン制1対1RPGの**戦闘システム(リソース制)**と**ゲーム進行(押し引きダンジョン→G獲得→ショップ→返済ノルマ→章/ボス→エンド)**を、
ブラウザ上で実プレイして検証するための自己完結HTML(依存なし)。アート・シナリオ等は含まない、**ロジックだけ**のサンドボックス。数値は仮値。

▶ **プレイ:** `index.html` をブラウザで開く(または GitHub Pages)。

## 進行(実ゲームと同等の骨格)
- **拠点(1章6日)**: 日を消費する行動=ダンジョン / モルガンで働く(即金・淫乱↑) / 宿。ショップは日を消費しない。
- **押し引きダンジョン**: 1階ずつ潜り、勝つたび稼ぎ**プール**が連勝倍率で膨張。各階後に「奥へ進む/引き上げる」。HP/EPは階持ち越し。
  **3・5階はボス雑魚(エリート)**=倒せば宝箱、負ければプール全損+淫乱↑。
- **ショップ**: G(ゴールド)で武具/防具/淫具/消耗品を買う。
- **返済日(7日目)**: 章ボスを倒す→ノルマを納付。足りなければ利子で繰越(最終章は破産)。全4章クリアでエンド(淫乱度で分岐)。

## 戦闘(リソース制・設計v2)
- **1手番=エネルギー(行動力)の予算内で複数行動**。行動は**攻撃 / スキル(強撃・崩し・見切り) / 防御(1ターン1回) / どうぐ**。
- **バフ・回復は「どうぐ」のみ**(消耗品=お金／発動型淫具=消耗なしだが使用でEP上昇)。
- **興奮(EP)が上がるほど最大エネルギーが減少**(−⌊EP/12⌋)→手数減→**絶頂敗北(EP100)**への死のスパイラル。
- 武具/防具が最大エネルギー±・特殊効果を持ち、ビルドが変わる。淫気解放は不採用。

## 「殴り合い脱却」の数値証明
下部の**オート対戦**で、同条件の **攻撃連打AI** と **賢いAI**(EPレース管理・大技を見切り)の勝率を比較。
**賢い＞連打** ならリソース管理に意味がある＝殴り合いを脱している。検証例: ボス雑魚(EPレース)で **連打43% / 賢い71%(+28)**、
ボス軽装(大技の脅威)で **+21**。
